# monorepo-newborn
***An inspirational project and base***

## 跑起来

`pnpm recursive install`		安装全部依赖

`dev:main-project`					启动前端服务

`dev:main-nodeapi`					启动后端node服务
`dev:examples`					    启动组件调试服务

`pnpm link @newborn/components`   进入examples 链接组件平台
